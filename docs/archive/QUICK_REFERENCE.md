# Quick Reference - Where Do Documents Go?

## Active Work (Put new docs HERE)
📁 `development/claude/`

- BUGFIX_*.md - Investigating bugs
- FEATURE_*.md - Building features  
- SESSION_*.md - Development notes
- fix needed.txt - Quick notes

## Completed Work (Move here when DONE)
📁 `archive/`

**Before moving:**
1. ✅ Update PROJECT_CONTEXT.md
2. ✅ Add to Issue Log (if bug fix)
3. ✅ Add summary at top of file

## Old Sessions (Move after 2+ weeks)
📁 `development/claude/ignore/`

Keep for historical reference.

---

## Naming Examples

```bash
# Today's date: 2025-01-06

BUGFIX_DUPLICATE_METHODS_2025-01-06.md
FEATURE_AUDIO_CAPTURE.md
SESSION_2025-01-06_BUILD_FIXES.md
PLAN_WEBCAM_INTEGRATION.md
```

---

## When to Update PROJECT_CONTEXT.md?

✅ After EVERY bug fix  
✅ After EVERY feature completion  
✅ After major refactoring  

❌ Not for work in progress  
❌ Not for tiny changes  

---

## Monthly Cleanup (1st of month)

1. Move docs >2 weeks old to appropriate place
2. Archive completed work
3. Update PROJECT_CONTEXT.md
4. Review and consolidate
